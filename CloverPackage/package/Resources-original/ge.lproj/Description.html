Clover v2 Ist open source basierend auf mehreren Projekten: Clover, rEFIt, XNU, VirtualBox.

Der Haupteil ist die letzte Version von EDK2. http://sourceforge.net/projects/cloverefiboot/


Hauptmerkmale:

- booten in 10.4 - 10.8 OSX, Windows EFI und Linux EFI

- Es ist ein EFI Bootloader der RuntimeServices für OS bereitstellt, das für Legacy Bootloader nicht möglich ist

- CLOVER kann die Standart Einstellungen selber konfigurieren. 

  Der Benutzer kann dies aber auch mit der config.plist selber einstellen

- Mit CLOVER kann man direkt über Startvolume in den Systemeindtellungen in ein anderes OS neu starten

- Die in config.plist geschriebene UUID wird dieselbe im OS sein

- SMBIOS wird auf Standard 2.6 angepasst

- ACPI wird auf Standard 4.0 korrigiert. DSDT wird von der gebooteden Partition geladen 

  oder vom EFI folder

- Anpassbare Festlegung von Neustart und Schlaf/Aufwach sowie PCIRootUID Wert

- Graphik Injektor für ATI, NVidia und Intel für eine große Anzahl von unterstützten Karten. 

  Auch anpassbar.

- USB fix (LegacyOff, Ownership, Builtin, clock-id) bis zu USB 3.0 

- HDA Injektor und Ethernet builtin Injektor

- CPU Turbo und P- und C-states Generator

- Laden von ACPI Tabellen (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- Laden von zusätzlichen Kexten und Sicherheits Modus für FireWire

- Standart boot mit Zeitverzögerung , anpassbare Oberfläche , sichern von Bildschirmfotos mit F10

- Sichern von boot.log im OS und preboot.log mit F2
