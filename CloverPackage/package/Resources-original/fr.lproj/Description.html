Clover v2 est open source base sur différents projets: Clover, rEFIt, XNU, VirtualBox.

Le projet principal est EDK2 ,dernière révision : http://sourceforge.net/projects/cloverefiboot/



Caracteristiques Principales :

- démarrer Mac OS X de 10.4 a 10.8 , Windows EFI et Linux EFI

- Clover est un EFI bootloader qui fournit un environnement pour différents systèmes exploitations impossible origine.

- Principalement Clover détecte automatiquement votre matériel et le configure par default. 

  Mais l' utilisateur peut le modifier en éditant le fichier config.plist

- avec Clover vous pouvez changer OS a partir du Startup Disque dans preference systeme

- UUID écrite dans le fichier config.plist sera la même qu' avec OSX

- SMBIOS au standard 2.6

- ACPI corrige au standard 4.0. Votre propre DSDT se charge a partir de la partition active 

  ou du dossier EFI.

- customisable RestartFix ainsi que corrections veille/réveil et valeur PCIRootUID

- Injecteur Propriétés Graphiques pour ATI, NVidia et Intel pour un large choix de cartes .. 

  Paramétrable également :

- USB fix (LegacyOff, Ownership, Builtin, clock-id) USB 2/3.0  

- HDA injecteur (son) et injecteur Ethernet integre

- CPU turbo and P- and C-states generator

- chargement des tables ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- chargement de pilotes supplémentaires (kexts)et mode sécurisé pour FireWire

- démarrage par default après timeout, interface modifiable GUI , et sauvegarde capture d'écran par GUI avec F10

- sauvegarde boot.log dans OS et preboot.log par GUI en tapant F2
