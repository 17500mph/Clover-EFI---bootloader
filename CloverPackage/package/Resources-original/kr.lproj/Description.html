Clover v2는 다음의 여러 프로젝트에 기반한 오픈 소스입니다: Clover, rEFIt, XNU, VirtualBox.

중심 프로젝트는 EDK2 최신 빌드입니다. http://sourceforge.net/projects/cloverefiboot/


주요 특징: 

Clover v2는

- OS X 10.4 - 10.8, Windows EFI, Linux EFI에서 부팅됩니다.

- EFI 부트로더로서 레거시 부트로더는 하지 못했던 RuntimeServices for OS를 제공합니다.

- 대다수 하드웨어를 자동 감지하여 모든 특성값을 디폴트로 설정해 줍니다. 

  하지만 사용자가 config.plist를 수정하여 그 값을 변경할 수 있습니다.

- 시동 디스크 prefPane을 이용하여 다른 운영체제로 재시작할 수 있습니다.

- config.plist에 설정된 UUID가 운영체제의 UUID와 동일합니다.

- SMBIOS가 2.6표준으로 정정됩니다.

- ACPI가 4.0으로 정정됩니다. 부팅 파티션이나 EFI 폴더에 있는 사용자DSDT를 읽어 옵니다.

- 재시작과 잠자기/깨우기, PCIRootUID값을 사용자가 수정할 수 있습니다.

- Graphics Properties Injector가  ATI, NVidia, Intel 등 다양한 VGA 카드를 지원하며

 사용자가 커스터마이징할 수도 있습니다. 

- USB 3.0까지 LegacyOff, Ownership, Builtin, clock-id 등을 고칠 수 있습니다. 

- HDA, 내장 랜 등을 위한 인젝터를 제공합니다.

- CPU 터보 기능과 P, C 스테이트 생성 기능을 지원합니다.

- ACPI 테이블을 읽어 들입니다. (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- 추가 켁스트를 읽어 오며, FireWire용 안전 모드를 제공합니다.

- 대기 시간 경과 후 자동 부팅, GUI 커스터마이징, F10으로 GUI 스크린샷 저장 가능합니다.

- GUI상에서 F2를 누르면 boot.log와 preboot.log파일을 저장합니다.
