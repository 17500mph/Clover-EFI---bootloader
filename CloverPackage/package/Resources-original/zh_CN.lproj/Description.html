Clover v2 是基于  Clover, rEFIt, XNU, VirtualBox 等不同专案的开放源码: 以  EDK2 为主要最新修正版. http://sourceforge.net/projects/cloverefiboot/


主要功能:

- 支持 引导 10.4 - 10.8 OSX, Windows EFI 及 Linux EFI

- EFI bootloader 提供 RuntimeServices 与 OS 运行 (传统引导器无法提供)

- 默认 自动侦测大部份的硬件并设置所有属性. 但是用户也可以在 config.plist 中修改

- 支持 启动磁盘 面板 设置其他 OS 重新开机

- 写入 config.plist 的 UUID 将与 OS 相同

- 修正 SMBIOS 至 standard 2.6 版本

- 修正 ACPI 至 standard 4.0 版本. 支持从 开机分区 或 EFI文件夹 加载 DSDT

- 支持 选用 修正重新开机 的错误

- 支持 睡眠/唤醒 修复 并 支持 PCIRootUID 设置

- 支持 ATI, NVidia 及 Intel  显示卡启用功能, 并支持客制化参数设置

- 支持 USB 及 USB 3.0 的修复 (LegacyOff, Ownership, Builtin, clock-id) 

- 支持 HDA 及 以太网路 注入功能

- 支持 开启 CPU turbo 及 自动生成 P-State 及 C-State 的功能。

- 支持 ACPI 表单加载 (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- 支持 加载额外 kexts 

- 支持 FireWire 安全保密模式  

- 支持 计时默认开机分区 及 客制化 图形用户界面 GUI

- 可按 F10 储存 图形用户界面 GUI 的截图

- 可按 F2 储存 OS 的开机纪录 boot.log 及 GUI 的开机纪录 preboot.log
