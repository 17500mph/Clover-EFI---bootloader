Clover v2 це UEFI завантажувач з відкритими кодами, заснований на зразках від Tianocore EDK2, і використовує технології проектів: Chameleon, rEFIt, XNU, VirtualBox.http://sourceforge.net/projects/cloverefiboot/


Основні можливості:

- завантаження систем OS X від 10.4 до 10.8, Windows EFI та Linux EFI

- Clover EFI надає системі Runtime Services, що неможливо для завантажувачів на основі BIOS.

- в основному Clover автоматично визначає склад комп'ютера і виставляє потрібні властивості за замовчанням, 

  але при бажанні користувач може встановити свої властивості в config.plist

- ви маєте можливість використовувати перезавантаження через Системні налаштування-> Стартовий Диск

- ви можете вказати свій UUID в config.plist, і будете впевнені, що саме його побачить система

- проводиться поправка таблиці SMBIOS до стандарту 2.6

- проводиться поправка таблиць ACPI до стандарту 4.0. Можна завантажити свою DSDT або з кореня завантажуваної системи, 

  або з розділу з завантажувачем

- коректування проблем рестарту та сну/пробудження

- автоматичне підключення відеокарт від ATI, NVidia і Intel. 

  з можливістю налаштування

- коректування проблем USB (до стандарту 3.0) 

- інжекція властивостей звукової та мережевої карт.

- управління живленням і частотами ЦПУ

- завантаження додаткових таблиць ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- завантаження додаткових kexts, встановлення режиму FireWire

- автозавантаження після паузи, налаштування зовнішнього вигляду (знімок екрану можно зберегти у файл по F10),

- збереження звітів як у системі, так і до запуску системи (по F2) для діагностики проблем.
