Clover v2 是基於  Clover, rEFIt, XNU, VirtualBox 等不同專案的開放源碼: 以  EDK2 為主要最新修正版. http://sourceforge.net/projects/cloverefiboot/


主要功能:

- 支持 引導 10.4 - 10.8 OSX, Windows EFI 及 Linux EFI

- EFI bootloader 提供 RuntimeServices 與 OS 運行 (傳統引導器無法提供)

- 預設 自動偵測大部份的硬體並設置所有屬性. 但是用戶也可以在 config.plist 中修改

- 支持 啓動磁碟 面板 設置其他 OS 重新開機

- 寫入 config.plist 的 UUID 將與 OS 相同

- 修正 SMBIOS 至 standard 2.6 版本

- 修正 ACPI 至 standard 4.0 版本. 支持從 開機分區 或 EFI文件夾 加載 DSDT

- 支持 選用 修正重新開機 的錯誤

- 支持 睡眠/喚醒 修復 並 支持 PCIRootUID 設置

- 支持 ATI, NVidia 及 Intel  顯示卡啓用功能, 並支持客制化參數設置

- 支持 USB 及 USB 3.0 的修復 (LegacyOff, Ownership, Builtin, clock-id) 

- 支持 HDA 及 乙太網路 注入功能

- 支持 開啓 CPU turbo 及 自動生成 P-State 及 C-State 的功能。

- 支持 ACPI 表單加載 (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- 支持 加載額外 kexts 

- 支持 FireWire 安全保密模式  

- 支持 計時預設開機分區 及 客制化 圖形用戶界面 GUI

- 可按 F10 儲存 圖形用戶界面 GUI 的截圖

- 可按 F2 儲存 OS 的開機紀錄 boot.log 及 GUI 的開機紀錄 preboot.log
