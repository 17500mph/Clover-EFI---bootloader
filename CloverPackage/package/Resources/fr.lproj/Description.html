Clover v2 est un programme open source basé sur différents projets: Clover, rEFIt, XNU, VirtualBox.

Le projet principal est EDK2 ,dernière révision ici : http://sourceforge.net/projects/cloverefiboot/



Caractéristiques Principales :

- démarrer Mac OS X de 10.4 a 10.8 , Windows EFI et Linux EFI

- Clover est un EFI bootloader qui fournit un environnement pour différents systèmes exploitations ce qui est impossible avec d' autres bootloaders.

- Clover détecte automatiquement votre matériel et le configure par default. 

  Libre a vous de modifier les paramètres en éditant le fichier config.plist

- Sous Clover vous pouvez changer OS a partir du disque de démarrage dans préférence système

- UUID écrite dans le fichier config.plist sera la même qu' avec OSX

- SMBIOS au standard 2.6

- ACPI actualisé au standard 4.0. Votre propre DSDT peut être pris en charge a partir de la partition active 

  ou du dossier EFI.

- RestartFix ,sleep/wake et valeur PCIRootUID personnalisable.

- Injecteur Propriétés Graphiques pour ATI, NVidia et Intel pour un large choix de cartes .. 

  Paramétrable également :

- USB fix (LegacyOff, Ownership, Builtin, clock-id) USB 2/3.0  

- injecteur HDA (son) et Ethernet intégré

- CPU turbo and P- and C-states generator

- chargement des tables ACPI (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- chargement de pilotes supplémentaires (kexts) et mode sécurisé pour FireWire

- démarrage par default après timeout, interface graphique modifiable GUI , sauvegarde de capture d'écran par GUI avec F10

- sauvegarde du fichier boot.log dans l 'OS et preboot.log par GUI en tapant F2
