Clover v2 adalah kompilasi open source dari: Clover, rEFIt, XNU, VirtualBox.

EDK2 revisi terbaru. http://sourceforge.net/projects/cloverefiboot/


Fitur-fitur:

- Untuk boot 10.4 - 10.8 OSX, Windows EFI dan Linux EFI

- Bootloader berbasis EFI yang menyediakan RuntimeServices buat OS dimana hal itu mustahil

  dilakukan oleh Legacy Bootloaders

- Clover akan mendeteksi perangkat keras yang ada secara otomatis dan mengatur semua properti

  secara default tapi user masih bisa merubahnya melalui config.plist

- Dengan Clover, anda dapat melakukan restart ke OS lain dari Startup Disk prefPane

- UUID yang tertulis didalam config.plist akan sama dengan yang ada dalam OS

- SMBIOS akan dikoreksi ke standard 2.6

- ACPI akan dikoreksi ke standard 4.0, DSDT custom akan dijalankan dari partisi yg dipakai boot 

  atau dari folder EFI

- RestartFix yang bisa diatur dan pembenahan untuk fitur sleep/wake, serta nilai PCIRootUID

- Injeksi kartu grafis untuk ATI, NVidia dan Intel dimana lebih banyak kartu yang didukung. 

  Dan inipun bisa dirubah disesuaikan keinginan kita.

- USB fix (LegacyOff, Ownership, Builtin, clock-id) sampai dengan USB 3.0 

- Injeksi untuk HDA dan Ethernet yang builtin

- CPU turbo dan dapat menghasilkan P- serta C-states.

- Memuat ACPI tabel (SSDT-xx, APIC, BOOT, SLIC, SLIT, SRAT, UEFI...)

- Menjalankan kext-kext ekstra dan modus aman untuk FireWire

- Boot setelah timeout, GUI yang bisa diatur, dan simpan screenshots GUI melalui tombol F10

- Simpan boot.log dalam OS dan preboot.log dari GUI melalui tombol F2
